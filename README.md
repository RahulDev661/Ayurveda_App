# Ayurveda_App
The Ayurveda App is a holistic wellness platform designed to bring the ancient wisdom of Ayurveda into modern daily life. It helps users maintain physical, mental, and spiritual balance through personalized health guidance, natural remedies, and lifestyle recommendations. 



# 🌿 Ayurveda Recommendation System

A full-stack web app that suggests Ayurvedic remedies based on symptoms.

## 🚀 Features
- Symptom-based disease prediction
- Ayurvedic remedies & herbs
- Full-stack (React + Node.js)
